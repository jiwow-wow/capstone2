👁️ Gaze-Based Focus & Drowsiness Detection System
MediaPipe Face Mesh와 OpenCV를 활용하여 사용자의 시선 추적, 눈 깜빡임 감지(EAR), 화면 내 객체 트래킹을 종합해 실시간 집중도를 평가하고 졸음을 경고하는 시스템입니다.

📂 파일 구조 및 역할
config_and_constants.py: 페이스 메쉬 초기화 및 EAR 임계값(0.25), 시선 민감도, 화면 이탈 기준(3.0초) 등 핵심 상수 관리

functions.py: 시선 좌표 계산, 웹캠 보정(Calibration), EAR 깜빡임 연산, Centroid 기반 객체 추적 함수 정의

main.py: 화면 캡처 및 웹캠 기반 메인 루프, 객체 탐지, 집중도 평가 알고리즘 및 결과 시각화 실행

주요 기능 (Core Features)
시선 추적 & 자동 보정: 초기 캘리브레이션 후 사용자의 얼굴 움직임이 감지되면 실시간으로 재보정하여 정확한 스크린 좌표를 매핑합니다.

졸음 감지: EAR(Eye Aspect Ratio)을 통해 눈 깜빡임을 측정하고, 10초당 평균 빈도가 기준치를 초과하면 졸음 경고(Drowsiness detected!)를 띄웁니다.

객체 트래킹: 화면 내 움직이는 객체를 탐지하고 고유 ID를 부여해 지속적으로 추적합니다.

집중도 평가: 시선이 객체에 머무르면 점수가 오르고, 객체를 외면하거나 화면 밖을 일정 시간(3초) 이상 바라보면 점수가 깎이는 패널티 시스템이 작동합니다.
